# SIBUR Store

Корпоративный магазин мерча для сотрудников и партнеров ПАО «СИБУР Холдинг».

## Структура проекта

- `frontend/`: Веб-приложение (Next.js 15, React 19, Tailwind CSS v4, TypeScript).
- `backend/`: (Планируется) API или статический JSON.
- `public/`: Статические ресурсы.
- `docs/`: Документация.

## Запуск разработки

### Frontend

```bash
cd frontend
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000).

## Стек технологий

- **Frontend:** Next.js (App Router), Tailwind CSS, TypeScript.
- **Дизайн:** Соответствует брендбуку СИБУР.
