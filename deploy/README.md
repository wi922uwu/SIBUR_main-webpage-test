# Инструкция по деплою SIBUR Store

## Предварительные требования
1. VPS на Ubuntu 20.04 / 22.04 / 24.04
2. Домен `sibur.store` направлен A-записью на IP сервера.
3. Код залит в публичный (или доступный по токену) репозиторий GitHub.

## Инструкция

1. Зайдите на VPS по SSH:
   ```bash
   ssh root@ВАШ_IP
   ```

2. Скачайте скрипт установки (если репозиторий публичный, можно просто клонировать один файл, но проще клонировать всё вручную первый раз, или создать файл скрипта).

   **Самый простой способ:**
   Создайте файл `setup.sh` на сервере и скопируйте в него содержимое из `deploy/setup.sh` этого проекта.
   
   ```bash
   nano setup.sh
   # Вставьте код, нажмите Ctrl+O, Enter, Ctrl+X
   chmod +x setup.sh
   ```

3. Запустите установку:
   ```bash
   ./setup.sh https://github.com/ВАШ_ПОЛЬЗОВАТЕЛЬ/ВАШ_REPO.git
   ```

4. Скрипт сделает всё сам: установит Node.js, Nginx, скачает код, соберет проект и запустит его.

## Обновление сайта

Когда вы запушите новые изменения в GitHub, зайдите на сервер и выполните:

```bash
cd /var/www/sibur-store/frontend
git pull
npm install
npm run build
pm2 restart sibur-store
```
